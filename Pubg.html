Pubg Mobile

Pubg Mobile is a multiplayer online game.In this game a player can team upto four members.The game logic is quite simple .The player must be a last man standing out of 100 players.The game is packed with Ar guns,Snipers,Grenades,Scope,and lot more.Currently,The game has 4 classic map which is Erangel,Miranmar,Sanhok and Vikendi.Each classic match max time is 30 minutes .Evo ground is also a most played.The team death match is the popular one than the rest.In the tdm match a player is added to a squad .There will 2 teams in the match.The team which gets 40 kills first will be the winning team .Pubg mobile has recently added pay mode.This mode has rocket launchers,helicopter,Bullet proof vehicle,grenade guns.This game is a very good time killer and will be more fun when played with friends.This games still now hold the craziness of gamers by regularly updating the game with new feature.
